Audit Importer

Audit Importer is a Google Sheets-based tool used to import audit data into structured audit worksheets.

The tool allows authorized users to paste audit output into a Google Sheet and automatically generate formatted audit reports. Audit Importer only accesses the spreadsheet in which it is installed and does not access Gmail, Google Drive files outside the active spreadsheet, or external services.
