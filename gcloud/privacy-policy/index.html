Audit Importer / Privacy Policy

Privacy Policy

Audit Importer is a Google Apps Script tool bound to a specific Google Spreadsheet. It helps authorized users import game audit output into structured audit worksheets inside that spreadsheet.

Google User Data Accessed

Audit Importer may access or interact with the following Google user data:

  • The active Google Spreadsheet where the tool is installed.
  • Sheet names, rows, columns, cell values, formatting, and template sheet content in that spreadsheet.
  • Audit data pasted by users into the import dialog, such as user IDs, usernames, kills, deaths, assists, or similar game audit fields.
  • Basic Google Apps Script execution information provided by Google, such as script runtime status and error logs.

How Google User Data Is Used

Audit Importer uses Google user data only to provide its spreadsheet import functionality:

  • It displays an "Import Audit" menu and dialog inside the active spreadsheet.
  • It reads the configured audit template sheet in the active spreadsheet.
  • It creates a new audit worksheet by copying the template sheet.
  • It writes the pasted audit rows into the new worksheet.
  • It preserves spreadsheet formatting needed to generate the formatted audit report.

OAuth Scopes Used

Audit Importer requests the following Google OAuth scopes:

  • https://www.googleapis.com/auth/spreadsheets.currentonly: used to read, copy, format, and write data only within the spreadsheet where the script is installed.
  • https://www.googleapis.com/auth/script.container.ui: used to add the "Audit Tools" menu and display the import dialog inside Google Sheets.

Audit Importer does not request access to Gmail, does not request broad Google Drive access, and does not access Google Drive files outside the active spreadsheet.

Data Storage

Audit Importer does not store data outside of the Google Spreadsheet where it is used and does not transmit audit data to external servers.

Imported audit data remains in the spreadsheet selected by the user. Any retention, deletion, sharing, or access control for the resulting audit worksheets is managed by the spreadsheet owner through Google Sheets sharing settings.

Data Sharing

Audit Importer does not sell, share, transfer, or disclose Google user data or pasted audit data to third parties. It does not use Google user data for advertising, analytics, profiling, or AI/ML model training.

Google API Services User Data Policy

Audit Importer's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
